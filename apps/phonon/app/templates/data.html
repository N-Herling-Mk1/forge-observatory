DATA PROVENANCE

where the phonon dataset came from, how it was found, and how to reproduce the pull

Origin

Dataset
Chen et al. (2021) DFPT phonon density of states — 1,524 inorganic crystals, each with a phonon DOS spectrum computed from density-functional perturbation theory.
Underlying source
The DFPT spectra trace to Petretto et al.'s high-throughput phonon database (Sci. Data 5, 180065, 2018); Chen et al. curated the 1,524-material learning subset.
Obtained from
github.com/zhantaochen/phonondos_e3nn  ·  branch main  ·  the authors' own reproduction repository.
How we found it
Via the paper itself — the Chen et al. data-availability statement links to this GitHub repo, which hosts both the curated .pkl and the train/test/val split file.
Date pulled
2026-06-24  (re-verified byte-for-byte 2026-06-25). The two core files were checked against recorded sizes and SHA-256 on re-pull.
License
See the upstream repository terms.

The paper

Citation
Chen, Andrejevic, Smidt, Ding, Xu, Chuang, Lou, Cheng, Owens, Li. Direct Prediction of Phonon Density of States with Euclidean Neural Networks. Adv. Sci. 8, 2004214 (2021).
Identifiers
arXiv:2009.05163  ·  DOI 10.1002/advs.202004214
What we reproduce
The E(3)-equivariant network (gate_points_2101) that maps a crystal structure to its 51-bin phonon DOS — our mk1 backbone, and the substrate for the FORGE Bayesian last-layer extension.

Composition

Materials
1,524 crystals · split 1,220 train / 152 val / 152 test (disjoint, covering all 1,524).
Target grid
51 DOS bins on a shared 0–1000 cm⁻¹ axis (20 cm⁻¹ step) — identical for every material. Targets are peak-normalized to [0,1].
High-res truth
A 4,404-point DFPT spectrum per material is also retained (Savitzky–Golay smoothed, window 101, order 3) for reference; the 51-bin learning target is the binned, clipped version.
Representations
periodic_crystal_graph (max_radius 5 Å) · mass_onehot_118 · dos_51bin_0_1000cm

File manifest

filebytesroleverified

Pipeline — structure to spectrum

CIF
pymatgen-generated crystal structure (list of CIF lines per material)
periodic graph
atoms → nodes, neighbors within 5 Å → edges with relative position vectors
mass one-hot
each atom tagged by Z in a 118-d one-hot (the only node feature)
51-bin DOS
E(3)NN regresses the phonon density of states on the shared 0–1000 cm⁻¹ grid
